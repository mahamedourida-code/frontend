# Button And Symbol Asset Prompts

## Generation Rules

- Use generated assets only for signature buttons and memorable product symbols. Build normal navigation, filters, cancel, sort, settings, and menu controls in Figma/code.
- Do not generate words inside buttons. Leave a clean empty center for text such as `Try it`, `Dashboard`, or `Upgrade` to be added in Figma.
- Button output: isolated horizontal button surface on a plain `#FFFFFF` background, 2048 x 1024, centered, generous clear margin, no surrounding UI.
- Symbol output: one isolated small object on a plain `#FFFFFF` background, 1024 x 1024, centered, no colored background box or tile behind it. Build the symbol container in Figma.
- Symbol direction: simple, unusual, recognizable objects rather than document pages or generic library icons. No words, no logos, no mockup interface.
- Button palette: deep green `#079765`, bright accent `#18E399`, lime upgrade accent `#D4FF6F`, charcoal `#0B111D`, neutral white.
- Symbol palette: neutral graphite, brushed silver, ivory, or muted gold with only one restrained AxLiner green accent.

## Key Button Surfaces

### 1. Primary Conversion Button

Use for: `Try it`, `Upload`, `Convert files`, `Start conversion`.

```text
Create a single blank horizontal SaaS call-to-action button surface, no text and no icon. Wide pill-rounded rectangle with a polished AxLiner green finish: deep emerald base #079765, slightly brighter green upper edge, subtle white reflected highlight along the top inner rim, very soft lower shadow, crisp vector-like 2D rendering. The button should feel premium and reliable, not playful, not glassy neon. Isolated on a pure white #FFFFFF flat background with large empty margin, straight front view, no perspective, no UI around it. Leave the center entirely blank for typography to be added later in Figma.
```

### 2. Account / Dashboard Button

Use for: `Sign up`, signed-in `Dashboard`.

```text
Create a single blank horizontal SaaS action button surface, no text and no icon. Wide pill-rounded rectangle in near-black charcoal #0B111D with a refined dark graphite upper gradient, thin pale reflected highlight on the upper inside edge, subtle inset depth and a clean small shadow below. It should look precise and confident, like a premium productivity product login action. Isolated on pure white #FFFFFF background, front view, large clear margin, no surrounding interface, center kept blank for a text label added in Figma.
```

### 3. Upgrade / Credits Button

Use for: `Upgrade`, `Buy credits`, quota reached prompts.

```text
Create a single blank horizontal upgrade button surface for a professional SaaS product, no text and no icon. Wide rounded rectangle in fresh lime-green: main fill #D4FF6F, lighter almost-white lime reflection across the top edge, gentle green-yellow depth toward the bottom, clean soft shadow. The result should feel valuable and inviting, not fluorescent or childish. Isolated on flat pure white #FFFFFF background with generous margin, front view, blank center ready for typography in Figma.
```

### 4. Reviewed Batch Download Button

Use for: the important post-review action `Download reviewed batch`.

```text
Create a blank wide action button surface for exporting a completed reviewed batch, no text. Use a dark charcoal base #0B111D, with a very thin emerald #18E399 accent line embedded along the lower edge and a restrained white upper highlight. Include one tiny empty reserved area at the left for an export symbol to be placed later, but do not draw an icon. The mood is final, secure, and professional. Isolated on #FFFFFF background, front view, no UI context, no glow.
```

## Simple Object Symbols

Figma supplies the card, hover state, and background. Generate only the object itself.

### Table Mode: Counting Frame

```text
Create one small isolated 2D symbol: a compact brushed-silver counting frame with six round beads arranged into clean rows, two beads in emerald green and the rest in charcoal. Front view, simple silhouette, slightly polished, memorable and professional. No paper, no numbers, no letters, no background tile, no shadow card. Place only the object on a flat pure white #FFFFFF background.
```

### Bank Statement Mode: Balanced Coins

```text
Create one small isolated 2D symbol: three neat muted-gold coins stacked on the left of a thin balance bar, with one emerald coin resting perfectly level on the right. Clean simple shape, no currency mark, no calculator, no text, no document, no surrounding box. Flat pure white #FFFFFF background.
```

### Invoice / Receipt Mode: Cash Drawer

```text
Create one small isolated 2D symbol: a compact charcoal cash drawer slightly open, showing three simple coin circles and one emerald divider inside. Modern rounded edges, minimal details, easy to recognize at small size. No receipt paper, no price tag, no words, no colored background box. Flat pure white #FFFFFF background.
```

### Text Output Mode: Fountain Nib

```text
Create one small isolated 2D symbol: a clean graphite fountain pen nib with one emerald ink drop beneath it. Symmetrical, elegant, simple, slightly polished, no letters, no lines of writing, no paper, no tile behind it. Flat pure white #FFFFFF background.
```

### Batch Upload: Token Stack

```text
Create one small isolated 2D symbol: four thin rounded tokens stacked with a slight offset, like a compact batch ready to move, with one emerald token between three neutral graphite tokens. Minimal object, crisp edges, no pages, no arrows, no cloud, no background container. Flat pure white #FFFFFF background.
```

Calm quality-check feeling, not an alert icon. No document, no exclamation mark, no text, no colored tile. Flat pure white #FFFFFF background.
