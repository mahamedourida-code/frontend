# AxLiner workspace symbol prompts

Transparent-background illustrations for the AxLiner accounting workspace — turning document chaos into clean books. Generate each with GPT Image 2 and drop the PNG into `/public/symbols/` using the exact **File** name.

> Every prompt below is **self-contained** — the full inline style sentence is pasted at the end of each one, so any single entry is copy-paste ready into GPT Image 2 without a separate reference token.
